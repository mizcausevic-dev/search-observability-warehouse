# Security Policy

## Scope

This repository contains sample warehouse data only. Do not commit real search console exports, customer crawl logs, production credentials, or warehouse connection strings.

## Reporting

If you find a security issue in the project structure or dependency setup, report it through GitHub Security Advisories or contact through [Kinetic Gain](https://kineticgain.com/).

## Usage Notes

- keep production data outside the repo
- rotate any credentials used for future warehouse integrations
- review dependency update PRs before merging
